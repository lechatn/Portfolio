PROJET IMMOBILIER

Analyse et prédiction des prix de l’immobilier

Date

Mars 2025 - Avril 2025

Technologies utilisées

Lien Github

OBJECTIFS DU PROJET

Ce projet a pour objectif de prédire les prix de vente des maisons dans le comté de King (USA) à l’aide d’une base de données réelle. L’enjeu principal était de créer un modèle de machine learning capable d’estimer le prix d’un bien en fonction de ses caractéristiques (surface, nombre de chambres, localisation, etc.).

Le projet comprend également le développement d’une API pour rendre les prédictions accessibles via une interface simple et réutilisable ainsi que la création de graphs pertinents pour nous aider durant le projet.

MON RÔLE DANS LE PROJET

J’ai été impliqué sur l'ensemble des parties du projet, de l’exploration des données à la mise en place de l’API. J’ai commencé par nettoyer et analyser les données avec Python (pandas), puis j’ai conçu un modèle de régression avec scikit-learn pour prédire les prix.

Ensuite, j’ai déployé une API avec FastAPI et Uvicorn afin de permettre à d’autres utilisateurs de tester le modèle facilement en ligne.

CE QUE J'AI APPRIS

Ce projet m’a permis d’approfondir mes compétences en data science, notamment en préparation de données, en modélisation prédictive et en interprétation de résultats.

J’ai également appris à concevoir et déployer une API avec FastAPI, à documenter proprement mon travail, et à rendre un projet data exploitable techniquement et compréhensible pour un public non technique.
